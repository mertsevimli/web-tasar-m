Css ile Arkaplan resmi


Elementlere arkaplan resmi tanımlamak için style="background-image:url()" özelliği kullanılabiliri.

Css ile Resim Konumu


Elementlere arkaplan resmi tanımlamak için style="background-image:url()" özelliği kullanılabiliri.

background-position: özeliği ile arkaplan resmini çerçeve içerisinde dilediğiniz konuma ayarlıyaibliriz.

Css ile Resim Sabitleme


Elementlere arkaplan resmi tanımlamak için style="background-image:url()" özelliği kullanılabiliri.

background-position: özeliği ile arkaplan resmini çerçeve içerisinde dilediğiniz konuma ayarlıyaibliriz.

background-attachment: fixed; ile arkaplan resmi sabitlenir.
