Css Position Özeliği

Html elementleri ekranda konumlandırmak için kullanılır

5 farklı position özeliği var

  1. static : varsayılan değer, eleman yerinde konumlanır..
  2. relative : göreceli
  3. fixed : istenilen noktaya sabitlenir.
  4. absolute : mutlak
  5. sticky : yapışkan
