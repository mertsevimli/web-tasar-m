Css Giriş Eğitimi

Css Nedir

Css: Cascading style sheet in kısatmasıdır. Ekrandaki html öğelerini tasarımlarını değiştirmemizi sağlar.

CSS yazma Seçenekleri

  1. Inline : Html elemenlerinin içerisinde style attribute ü açılarak yazılır.
  2. Sayfa head alanını içerisinde style etiketleri araasında yazılarak
  3. Harici bir css dosyası oluşturup sayfa head alanı içinde çağırarak.

Inline Css Yazılımı

Elementlerin açılış taglarına boşluk bırakıp style özeliği içine css kodu yazarak yaparız.
eğer birden fazla css özeliği uygulayacaksak özellikler arasında ; koyarız. En sondakine ; koyamaya gerek yoktur.

Head Tagları Arasında CSS Yazılımı

Sayfa head alanı içerisinde style etikeleri arasında yazılarak sayfadaki elementlere stil uygulanabilir.

Harici CSS Dosyası Kullanarak Tasarım

Harici bir css dosyası oluşturup (style.css ) sayfa fhead alanı içinde link özeliği ile çağırarak sayfa tasarımını şekillendirebiliriz.

CSS Seciciler

1- Element Seciciler

Sayfadaki elementleri html taglarına göre (p, h1 , vb) secerek css uygulayabiliriz.
Bu durumda sayfadaki tüm elementler seçilecektir.

2-Id Seçici

Sayfadaki elementleri id özelliğine göre seçip stil uygulamamızı sağlar.
Genelde sayfadaki tek bir elementi seçmek için kullanırız.
Css dosyası içerisinden # karakteri yazılıp elementlerin id değeri eklenerek element seçilir.
Örnek : #idSecici()

3-Class Sınıc Secici

Web sayfasında veya uygulama genelinde birden fazla elemente aynı stil ayarlarnı uygulamak için class secicileri kullanınırız.

Element tagı içerisinde class yazıp css dosyasında yazılan class adını vererek kullanırız.

Css dosyası içerisinde . işaretiyle class oluştururuz.
Örneğin : .sinifSecici{}

Bir elemente clas özeliği içerisine aralarına boşlık birakarak birden fazla class uygulayabiliriz.

Ana div içindeki ilk p

Bir html elementinin içindeki alt elementleri hedefleyerek de stil uygulayabiliriz.

Bunun için ana elementi seçip boşluk veya > işaretiyle alt elementleri seçebiliriz. Örneğin : #anaDiv p{}

anaDiv içindeki son p elementi

css ile Elementlere Kenarlık (border) uygulama

Sayfadaki elementlere 4 yönden (üst-sağ - alt-sol) ayrı ayrı border uygulanabilir.
Tek bir border elementiyle 4 yönede border uygulanabilir.

Border solid : düz çizgi

Border dashed : kesik çizgi

Border dotted : noktalı çizgili

Border double : Çift çizgi

Border inset : içeri doğru çizgi
Border outset : Dışarı doğru çizgi
