Bootstrap Icons- Web uygulamalarında küçük ikonlar kullanabilmemizi sağlayan kütüphane

Kullanımı :

  1. Bootstrap sitesinden icons menüsüne gidiyoruz.
  2. Aşağıda install menüsüne tıklıyoruz
  3. Uygulamızda hızlıca kullanmak için CDN kodunu direkt kopyalayabilriiz.
  4. Download ile bilgisayara indirip kütüphanesi projeye yükleyerek kullanabiliriz.
  5. CSS kütüphanesini ikon kullanacağımız ekranlarda sayfanın head tagları arasında çağrıyoruz.
  6. Sonrasında Bootstarp icons menüsündeki ikonlara tıklayıp açılan sayfadan i elementini class ile birlikte kopyalıyoruz.
  7. Kopyaladığımız ikonu sayfamızda kullamaz istediğimiz yere yapıştırıyoruz.
