Arama Sonucları

http://127.0.0.1:5500/search.html?Kelime=Bilgisayar bu url de search.html acıklaması istenen sayfa adresidir.
sonrasında ki ? işareti bu sayfaya bir query string geleceğini ifade eder
Query string ise sayfadan sayfaya adres çubuğu kullanılarak veri taşımamızı sağlar.
query string 2 alandan oluşur key - value
Bu örnekte (Kelime = Bilgisayar) kelime key, bilgisayar valuedir.
Bizde bu key değerine ait value ve ulaşarak bu değeri veritabanı vb aratıp ekranda cıktı oluşturabiliriz.
Query String de 1 den fazla parametre de taşıyabiliriz.
Bu durumda parametrelerin arasında & işareti konulur.
Örneğin : ?kelime=bilgisayar¶metre2=değer2 gibi.
